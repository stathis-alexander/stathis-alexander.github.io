# stathis-alexander.github.io

Personal website.

## Developing

Manages tools with `asdf`.

```bash
asdf install
```

Uses `bun`. Hosts at `localhost:9000`.

```bash
bun run dev
```

### Adding shower thoughts

1. Add a new markdown file to `shower-thoughts/src`, the filename should be `YYYY-mm-dd-a-title.md`.
2. Compile with `bun run build`.
3. Commit & push.
